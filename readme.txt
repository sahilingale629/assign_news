## Installation Instructions
To set up the Daily News Application locally, follow these steps:

### Prerequisites

Make sure you have the following installed on your machine:

- **Node.js**: You can download it from [Node.js Official Website](https://nodejs.org/).
- **npm** (Node Package Manager): This is included with Node.js.

### Step-by-Step Installation

1. **Clone the Repository**

   Open your terminal or command prompt and run the following command to clone the project repository:

   ```bash
   git clone https://github.com/sahilingale629/assign_news.git

2. **Install Necessary Dependencies**
   ```bash 
   cd assign_news
   cd backend
   npm install

3. **Start the Server**
    ```bash 
    npm start

4. **Open port 3000 on Browser**
   ```Browser
   http://localhost:3000/
